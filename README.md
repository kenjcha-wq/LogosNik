LogosNik | 文学实验室 (Literary Laboratory) v16.2“在拟演中触达真实，在合议中重构表达。”LogosNik 是一个专为深度创作者、哲思者设计的数字化写作环境。它打破了传统 AI 写作工具“单点问答”的局限，构建了一个模拟“编辑部合议制”的并行创作系统。🖋️ 核心创新：为什么它与众不同？1. 四机位人格合议 (Multi-Station Harmony)LogosNik 不再仅仅是一个对话框，它是一个并行的审计系统。当你完成一段草稿，可以一键启动“合议”，调用四个独立配置的 AI 人格：文学 (Literary)：侧重文风润色与感官描写，捕捉村上春树式的疏离感。联网 (Search)：实时检索事实，确保内容的准确性与本体论逻辑。哲学 (Philosophy)：基于存在主义逻辑进行审计，指出思维中的“自离”与漏洞。流行/定制 (Trendy/Custom)：优化排版与标题张力，适配不同传播场景。2. 模型主权解耦 (Cloud/Local Decoupling)真正实现了“插拔式”的模型调用架构。每个通道都可以独立设置：云端模式：支持 DeepSeek、Kimi、Qwen、MIMO 等所有兼容 OpenAI 格式的 API。本地模式：无缝对接本地算力（如 Ollama），让私密的哲学推演在本地运行，确保绝对的数据主权。双预设存储：每个机位都预设了云端/本地两套配置，一键切换，无需重复输入。3. “拟演式”空间构建我们认为创作需要一种“仪式感”：Notebook Paper 意向：中间主编辑区采用横格纸设计，缓解数字化屏幕的焦虑。地理坐标与时间轴：顶部实时显示经纬度坐标 (36°03'N 103°48'E) 与时钟，赋予虚拟创作一种真实的地理存在感。侧边参考窗：右侧内置哲学资料库与上传文档面板，实现人机协同参考。📱 极致适配电脑端：经典“三位一体”学者书斋布局。手机端：深度优化的侧滑抽屉（Swipe Drawer）交互，压缩上下边距以释放最大写作空间，支持在园子、店里或旅途中随时记录灵感。🛠️ 技术栈架构：纯原生 JavaScript (Single HTML File)，零依赖，极致轻便。样式：Tailwind CSS + 响应式布局。字体：Noto Serif SC (思源宋体) + Noto Sans SC，确保文学质感。存储：浏览器本地存储 (LocalStorage) + 自定义云同步接口。🚀 快速开始下载本项目唯一的 Js1.5.html 文件。使用浏览器打开。在 ⚙️ 设置 中配置你的 API Key 或本地模型地址。开始你的拟演创作。LogosNik 不试图取消创作过程，它试图强化创作深度。
